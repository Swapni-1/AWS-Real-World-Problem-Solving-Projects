2. Subnets - Division of one big network. Division is done so that it can be manageable. 
  -- In aws these are the Ip classes that are use 
   - 10.0.0.0 (most used ip class) | Range - 10.0.0.0 to 10.255.255.255
   - 173.16.00 | Range - 173.16.00 to 173.31.255.255
   - 192.168.0.0 | Range - 192.168.0.0 to 192.168.255.255
  -- CIDR Block 
   - ipv4 address if of 32 bit
   - 10.0.0.0/16 that means first 16 bit block are for network (10.0) and it cannot be changed
     other part last 1st and 2nd (0.0) are for host and it can be range from 0 to 255-2=253 
   - ipv4 address with cidr block should not overlap each other that means subnet-a 
     and subnet-b whatever sets of IPs they contains should not match each other , they should have
     entirely different sets of IPs.
     * Overlapping Example = 10.0.1.0/16 and 10.0.1.1/16
     * Non-Overlapping Example = 10.0.1.0/16 and 10.0.2.0/16
